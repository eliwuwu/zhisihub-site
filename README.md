# 织思 Hub 官网

织思 Hub 是面向长期项目、资料整理和 AI 协作的思考中枢。

## 本地预览

```bash
python3 -m http.server 4173
```

打开 `http://localhost:4173`。

## 文件结构

- `index.html`：首页
- `styles.css`：样式
- `script.js`：演示切换与内测表单本地反馈
- `privacy.html`：隐私政策
- `terms.html`：服务条款
- `robots.txt` / `sitemap.xml`：搜索引擎基础文件
