## 当前可用方案

1. GitHub Pages 稳定链接已经可用：`https://eliwuwu.github.io/zhisihub-site/`。
2. 网站结构已从“织思单产品页”改为“White Matter 白质工作室官网”。
3. `zhisihub.com` 后续再做 DNS 解析到 GitHub Pages 或国内托管平台；解析完成前不在仓库中放置 `CNAME` 文件，避免默认访问链接跳转失败。

## 国内正式访问路径

1. `zhisihub.com` 保留在腾讯云域名账号下。
2. 如果要使用中国大陆服务器或大陆 CDN 正式提供访问，需要准备 ICP 备案。
3. 备案未完成前，可先使用 GitHub Pages、香港节点、腾讯云 EdgeOne Pages、Cloudflare Pages 或 Vercel 类静态托管做演示。
4. 备案完成后，推荐迁移到腾讯云静态网站、COS、EdgeOne 或轻量应用服务器，并把 DNS 解析到国内服务。
5. 正式收集合作意向时，把当前本地表单改为飞书多维表、腾讯云云函数、企业微信或自有后端。
